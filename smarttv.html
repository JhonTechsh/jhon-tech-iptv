Tutoriais IPTV para Smart TV e Dispositivos

Este tutorial é válido para todas as Smart TVs e dispositivos compatíveis com IPTV. Selecione abaixo o modelo mais próximo do seu aparelho.

Samsung Smart TV (Tizen)

Aplicativo: IPTV Smarters Player

Samsung Apps → Instalar → Login com Xtream Codes → Inserir dados

LG Smart TV (WebOS)

Aplicativo: IPTV Smarters Player

LG Content Store → Instalar → Login com Xtream Codes → Inserir dados

Android TV (Sony, TCL, Philips, AOC, Xiaomi)

Aplicativo: XCIPTV Player

Play Store → Instalar → Xtream Codes → Inserir dados

TV Box Android

Aplicativo: XCIPTV Player

Play Store → Instalar → Xtream Codes → Inserir dados

Amazon Fire TV Stick

Aplicativo: XCIPTV Player

Appstore → Instalar → Xtream Codes → Inserir dados

Chromecast com Google TV

Aplicativo: XCIPTV Player

Play Store → Instalar → Xtream Codes → Inserir dados

Roku TV

Aplicativo: IPTV Smarters (Canal Roku)

Adicionar canal → Inserir dados via painel

Apple TV (tvOS)

Aplicativo: IPTV Smarters Player

App Store → Instalar → Xtream Codes → Inserir dados

TVs Antigas (sem loja de apps)

Solução: Usar TV Box ou Fire Stick

Conectar dispositivo → Instalar app → Login IPTV
