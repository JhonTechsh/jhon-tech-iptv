Tutoriais IPTV para Smart TV

Escolha abaixo a marca da sua TV para visualizar os aplicativos compatíveis e o passo a passo de instalação.

Samsung Smart TV (Tizen)

Apps compatíveis: Assist Plus, IBO Player, IBO Pro

Assist Plus: App pago, porém gratuito para clientes Jhon Tech.

Samsung Apps → Instalar Assist Plus → Abrir → Anotar código → Enviar no WhatsApp

IBO Player / IBO Pro:

Samsung Apps → Instalar → Abrir → Anotar Device ID/MAC → Enviar para ativação

LG Smart TV (WebOS)

Apps compatíveis: Assist Plus, IBO Player, IBO Pro

Assist Plus:

LG Content Store → Instalar → Abrir → Anotar código → Enviar para Jhon Tech

IBO Player / IBO Pro:

LG Content Store → Instalar → Abrir → Anotar Device ID/MAC → Enviar

Android TV / TV Box / Fire Stick

Apps compatíveis: XCIPTV Player, IBO Player, IBO Pro

XCIPTV Player:

Play Store → Instalar → Xtream Codes → Inserir usuário, senha e servidor

IBO Player / IBO Pro:

Play Store → Instalar → Abrir → Anotar dados → Enviar para ativação

Roku TV

Apps compatíveis: Assist Plus, IBO Player

Assist Plus:

Adicionar canal → Abrir → Anotar código → Enviar para Jhon Tech

IBO Player:

Adicionar canal → Abrir → Anotar Device ID e Key → Enviar para ativação

Apple TV (tvOS)

Apps compatíveis: IPTV Smarters Player, IBO Player

App Store → Instalar app → Xtream Codes ou ativação via ID → Inserir dados
