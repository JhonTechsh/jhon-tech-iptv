Tutoriais IPTV para iPhone (iOS)

Este tutorial é válido para iPhone e iPad. Antes de iniciar, certifique-se de estar conectado à internet. Após contratar o plano, você receberá os dados de acesso no WhatsApp.

IPTV Smarters Player Lite (GRATUITO)

Passo 1: Abra a App Store no seu iPhone.

Passo 2: Pesquise por IPTV Smarters Player Lite e instale o aplicativo.

Passo 3: Após a instalação, abra o aplicativo.

Passo 4: Aceite os termos de uso.

Passo 5: Selecione Login with Xtream Codes API.

Passo 6: Preencha os dados enviados pela Jhon Tech:

  • Usuário
  • Senha
  • URL do servidor

Passo 7: Toque em Add User e aguarde o carregamento.

Após o login, os canais, filmes e séries aparecerão automaticamente.

GSE Smart IPTV (GRATUITO)

Passo 1: Abra a App Store.

Passo 2: Pesquise por GSE Smart IPTV e instale.

Passo 3: Abra o aplicativo após a instalação.

Passo 4: No menu inicial, toque em Xtream Codes API.

Passo 5: Insira os dados fornecidos pela Jhon Tech:

  • Usuário
  • Senha
  • URL do servidor

Passo 6: Salve e aguarde a sincronização.

O conteúdo será carregado automaticamente.
